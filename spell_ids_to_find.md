# Missing Spell IDs Search List

Total missing: 71

## Instructions
1. Click each WoWHead link to find the spell
2. The spell ID is in the URL: `https://www.wowhead.com/spell=12345` (12345 is the ID)
3. Use `ruby _scripts/update-ids.rb [class] [key] [spell_id]` to add each ID
   Example: `ruby _scripts/update-ids.rb warrior e 6572`
